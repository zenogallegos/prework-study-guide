Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • typing 'git add -A' will commit all files to the working branch
  • 'git commit -m ""' will commit the changes to the repository
  • always use 'git pull origin main' before pushing to pull the base branch into your current branch
  • use 'git push origin (name of branch)' to push your commit
  • update the local repository by using 'git checkout main' and then 'git pull origin main'
